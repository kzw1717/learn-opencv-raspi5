# 用語解説：リスト内包表記

> `leaf_monitor.py` を読んでいて、`[c for c in cnts if cv2.contourArea(c) >= MIN_LEAF_AREA]` のような角かっこの中に for と if が入った書き方に出会います。
> これが**リスト内包表記**です。最初はギョッとしますが、考え方が分かればむしろ読みやすい、とても Python らしい書き方です。

---

## 1. これは何？（考え方）

リスト内包表記は、ひとことで言うと

> 「あるリストから、**条件に合うものだけ／加工したものを集めて**、新しいリストを1行で作る」

書き方です。

たとえば「たくさんの輪郭の中から、面積が一定以上の塊だけを集める」。これは第6章でやった、ノイズを除いて本物の塊だけ残す処理そのものです。ふつうの `for` 文でも書けますが、リスト内包表記なら短く・読みやすく書けます。

---

## 2. 基本の形

形は2種類だけ覚えれば十分です。

```python
# (A) 全部を加工して集める
[式 for 変数 in もとの並び]

# (B) 条件に合うものだけ集める
[式 for 変数 in もとの並び if 条件]
```

「**もとの並びを順に見て（for）、（条件に合えば）式の結果を集める**」と読みます。`leaf_monitor.py` で使われているのは、主に (B) の「条件でふるい分ける」形です。

---

## 3. サンプルスクリプト

まずは数字で、`for` 文との対応を見てみましょう。

```python
# sample_listcomp.py

numbers = [10, 50, 600, 30, 900, 700]

# ── ふつうの for 文で「600以上だけ集める」 ──
big = []
for n in numbers:
    if n >= 600:
        big.append(n)
print("for文 :", big)

# ── 同じことをリスト内包表記で ──
big2 = [n for n in numbers if n >= 600]
print("内包表記:", big2)

# ── (A) 全部を加工する例：2倍にする ──
doubled = [n * 2 for n in numbers]
print("2倍   :", doubled)
```

実行結果：

```
for文 : [600, 900, 700]
内包表記: [600, 900, 700]
2倍   : [20, 100, 1200, 60, 1800, 1400]
```

`for文` と `内包表記` の結果が同じであることに注目してください。**同じ処理を短く書いているだけ**です。

---

## 4. 代表的な使い方

- **ふるい分け**：条件に合う要素だけ残す（`if` を付ける）。`leaf_monitor.py` の主な用途。
- **加工**：全要素を変換して新しいリストにする（`if` なし）。
- 上の2つの組み合わせ：`[n * 2 for n in numbers if n >= 600]`（600以上のものを2倍にして集める）。

> **やってみよう**
> `sample_listcomp.py` の `numbers` を自由に変えて、「100より大きいものだけ集める」「全部を10で割る」などを内包表記で書いてみましょう。

---

## 5. leaf_monitor.py での実例

`leaf_monitor.py` には、リスト内包表記が2か所出てきます。どちらも「輪郭の中から、面積で本物だけをふるい分ける」用途です。

**(1) マットの断片をふるい分ける** — `mat_region_from_mask()`

```python
pts = [c for c in cnts if cv2.contourArea(c) >= MIN_MAT_FRAGMENT]
```

「輪郭 `cnts` を順に見て、面積が `MIN_MAT_FRAGMENT` 以上の `c` だけ集める」。小さなゴミをマットの断片と勘違いしないための足切りです。

**(2) 堆積物の塊をふるい分ける** — `analyze()`

```python
clumps = [c for c in leaf_cnts if cv2.contourArea(c) >= MIN_LEAF_AREA]
```

「堆積物の輪郭 `leaf_cnts` を順に見て、面積が `MIN_LEAF_AREA` 以上の `c` だけ集める」。影やノイズの小さな塊を、塊数の数えから除いています。

どちらも、第6章の「面積でふるい落とす」をそのまま1行にしたものだと分かります。

---

## 6. まとめ

- リスト内包表記は「もとのリストから、条件に合うもの／加工したものを集めて新しいリストを作る」書き方。
- 基本形は `[式 for 変数 in 並び]` と `[式 for 変数 in 並び if 条件]` の2つ。
- `for` 文で書いても同じだが、内包表記のほうが短く読みやすい。
- `leaf_monitor.py` では「面積で輪郭をふるい分ける」のに使われている。
